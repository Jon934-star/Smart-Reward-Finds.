# Smart Reward Finds

A trust-first reward and offers blog built with Astro.
Curated gift card giveaways, smart tools, reward apps,
and beginner-friendly guides.

**Live site:** https://smartrewardfinds.vercel.app

## About
Smart Reward Finds helps everyday people discover
legitimate gift card platforms, reward apps, smart tools,
and trusted offers — explained clearly for beginners.

Legit Finds. Real Rewards. Smarter You.

## Tech Stack
- Astro (static site framework)
- Deployed on Vercel
- Blog posts written in Markdown
- No database required

## Adding a New Blog Post
1. Go to src/content/blog/
2. Create a new .md file
3. Add frontmatter at the top
4. Write your post
5. Commit — site updates automatically
